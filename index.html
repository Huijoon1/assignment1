Project Idea

Overview

The IT project I am thinking of is a facial mask detection. By using a computer software, I would like to detect people devoid of masks. This project can be used with CCTV placed in public places such as movie theaters, offices, or shopping malls. The software I will be using is Cifar-10 Image classification software, and it can be used with deep learning algorithms. CNN, Convolutional Neural Network learning technique will also be used to detect people's facial information and extract those images and send those information immediately. By detecting such people with no masks, it will be very useful to prevent the spread of COVID-19 in public places and end this long pandemic.

Motivation

Since the COVID-19 pandemic, our lifestyle has changed, and it has become our daily routine to go out wearing facial masks. In order to prevent the spread of COVID-19, wearing facial masks becomes very crucial that many governments and organizations are ensuring the usage of facial masks. Because still a lot of people walking outside do not wear masks these days, I am aiming to detect people devoid of masks. According to OWID, there are an average of three million COVID-19 confirmed cases per day, which means the spread does not seem to be disappeared. This is the motivation of the IT project.

Description

Facial mask detecting project is a project to determine whether or not someone is wearing a mask. In this COVID-19 pandemic, wearing facial masks has become our daily routine not to spread the virus, so this project is crucial to prevent the spread. there are various object detection systems introduced. For example, in medical applications, deep learning algorithms are widely used, and in predicting crime rates, by using machine learning and the linear regression algorithm, it is possible to predict the percentage of crimes that can occur in the future. Deep learning architectures have recently demonstrated a significance in object detection. These architectures can be used to detect a mask on a person's face. Thus, this IT project's goal is to detect facial information, to determine the presence or absence of a facial mask and to prevent the future spread of the virus.

To begin with, CCTV cameras are utilized to gather real-time video footage of various public locations throughout the city. Facial images are taken from the camera clip, and these images are used to identify the mask on the face. The learning technique Convolutional Neural Network (CNN) is used to extract features from images, and subsequently numerous hidden layers learn these features. When the architecture detects people without a face mask, the information is sent to the appropriate authorities via the city network, who will take the appropriate action. The proposed system evaluated promising output based on data gathered from various sources. In this pandemic case, we also represented a system that can assure adequate law enforcement on persons who are not following fundamental health requirements.

So, as I said above, the Cifar-10 dataset and keras library of Python, and the Cifar-10 dataset is a widely used computer vision dataset for image recognition. It's a subset of the 80 million tiny pictures dataset, comprising 60,000 3232 color images divided into 10 item groups, each with 6000 photos. There are 50000 photos for training and 10,000 images for testing. And keras is a Python-based neural network library that is open-source. It has the ability to run on top of TensorFlow, which makes things easier for us. In comparison to tensorflow and other deep learning frameworks, Keras is a lot easier to use. This is why keras is being used in this project.

Deep learning systems can classify images fairly correctly. In this project, we'll use the Cifar-10 dataset and Python's keras package. Ten types of photos in the cifar-10 dataset. We can use keras to design a CNN model and submit the dataset for training. With those various types of images and photos, it will be easier to classify people's faces and images.

Also, this IT project's goal is to make an automatic system which predicts and offers accurate and timely information on people's information. We utilize machine learning, genetic, soft computing, and deep learning methods to evaluate massive data for systems. These algorithms have the potential to be extremely effective in predicting correct data and images with minimal complexity.
